SpaceX Missions App
======
Simple Application fetching data from (https://api.spacex.land/graphql/) api with option to add favourite missions.

### Tech stack
Core - `React` <br />
Language - `JavaScript ES6` <br />
Main React Libraries: <br />
- `react-router-dom` <br />
- `redux` <br />
- `react-persist` <br />
- `graphql` <br />
- `apollo` <br />
- `styled-components` <br />
- `material-ui` <br />
